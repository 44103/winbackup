# Win Backup
バックアップバッチ

## 使い方
1. バックアップしたいフォルダ内に `backup.bat` ファイルを配置
2. [optional] バックアップ対象外のファイル・フォルダを指定
   1. `bakignore.txt` ファイルを `bat` ファイルと同じフォルダ内に配置
   2. バックアップ対象外のファイル・フォルダを記載
      - ファイル名は拡張子まで記載
      - フォルダ名の後に `/` を記載
      ```
      ignore_file.xlsx
      ignore_folder/
      ```
3. `backup.bat` をダブルクリックして起動
   - `bak` フォルダが作成され、ファイル名の末尾に日付が追記される
